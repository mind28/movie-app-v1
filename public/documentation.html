MoviePop

This app will allow users to search for popular movies and add their top movies. This app will also allow users to search movies by genre, actor, director etc

Request URL HTTP Method Request Body Data Format Response Body Data Format
Create a movie and add it to the Movies Collection /movies POST A JSON object holding the movie information.
{
"Title": "Test",
"Description": "abc",
"Genre":{
"Name": "Drama",
"Description": "abc"
},
"Director": {
"Name":"Robert Zemeckis",
"Bio": "abc"
},
"Actors": ["Mike Jordan"],
"ImagePath": "https://picture.jpg",
"featured": false
} 		A JSON object holding data about the movie created
Add an actor to a movie and Update to Movies Collection /movies/:Title PUT {
"Actors": ["Mike Jordan"],
} 		A JSON object holding data about the movie that was updated
list all movies /movies GET none A JSON object holding data about all the movies
Get data about a single movie by title /movies/:title GET none A JSON object holding data about single a movie containing description, genre, director, image URL, whether its featured or not
Get data about a genre (description) by name/title (e.g. "Action/Adventure") /movies/:genre_name GET none A JSON object holding data about a genre
Get data about a single a dicector (bio, date of birth, death year) by name /movies/:director_name GET none A JSON object holding data about a director
Delete a movie from the Movies Collection /movies/:Title DELETE none A success message indicating that the movie was removed
Add a new user /users POST A JSON object holding the user information.
{
"user_name": "JaneDoe123",
"email": "JaneDoe123@mail.com",
"password": "123456!",
"date of birth": "01.01.1990"
} 		A success message indicating that the profile was created
List all users from the Users Collection /users GET none A JSON object holding data about all the users
Update a users information (username) /users/:id PUT none A success message indicating that the profile was updated
Allow users to add movies to their list of favorites /users/:id/:movieTitle POST A JSON object holding data about the movie to add A success message indicating that the movie was added
Allow users to remove a movie from their list of favorites /users/:id/:movieTitle DELETE none A success message indicating that the movie was removed
Allow users to deregister /users/:id DELETE none A success message indicating that the user has been removed
